Welcome to the Dodododo server world download website!

A few things you need to know: These world zips are in Paper Minecraft (a very efficient server platform) dimension design. This means that the nether and end dimensions are in seperate folders (e.g., DisasterSMP_the_nether and DisaserSMP_the_end). In order to make these files readable by Minecraft, you'll need to rename [worldname]_the_nether to "DIM-1" and [worldname]_the_end to "DIM1". Next you move both the DIM-1 and DIM1 folders into the [worldname] folder. Lastly, move this [worldname] folder into your .minecraft/saves folder. You should be able to find the world somewhere in your world list.

Click on a world description to start the download. NOTE: some of these files (particularly the amplified one) are extremely large, so it may take awhile
